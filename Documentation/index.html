class codeInterp.tester¶

The main class for the game logic.

Attribute objList:
 Stores a list of game objects form Blender.
Attribute scene:
 Stores the scene object of the current Blender level.
Attribute codeList:
 Stores all the code that a player submits per level.
Attribute levelScore:
 Keeps track of user score per level.
Attribute staticPosX:
 Static X world position for player avatar default.
Attribute staticPosY:
 Static Y world position for player avatar default.
Attribute staticPosZ:
 Static Z world position for player avatar default.
Attribute actions:
 Array for storing list of actions to be performed on user submission.
Attribute actionsLen:
 Keeps track of total action list length for submission.
Attribute step:Progression of current submission.
Attribute winObj:
 String name of the Blender object that the player needs to reach.
Attribute val:Bool to ensure player is on an object.
Attribute obj_name:
 Static string that stores the name of the player object.
Attribute text:Stores the text (code) that a user submits to be evaluated.
Attribute m:Initialise the mover class.
Attribute p:Initialise the picker class.
Attribute running:
 Keeps track of whether the player code is interpreted and executing.

Lerp function to move the player avatar object along with the main game loop.

Parameters:
  • startPos (list) – The position of the player avatar object before the move.
  • endPos (list) – The position that the player avatar object needs to move towards.
  • percent (float) – The time delta as calculated in moveUnitOne.

The position of the game object.

moveUnitOne(obj, dir, winObj)¶

Function to manage the movement of the player avatar and check collisions.

Parameters:
  • obj (KX_GameObject) – The Blender game object of the player avatar.
  • dir (str) – The string which indicates the direction that the avatar will move.
  • winObj (KX_GameObject) – The Blender game object of the object that needs to be conected with in order for the player to win.
Returns:

-1 if the player avatar object fails a raycast test, 2 if the player object raycasts and moves towards the winning object, the time delta of the movement.
